## Project Description

We developed a new version of **SocialCalc** using **Node.js and ReactJs**. This update modernized the underlying technology, enhancing performance, security, and alignment with current web development standards. Our project retained the core functionalities of SocialCalc while introducing new features such as real-time collaboration and a more intuitive user experience.
